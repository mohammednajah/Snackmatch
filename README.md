SnackMatch is a web application that helps users find and share their favorite snacks with friends and family. Whether you're looking for new snack ideas or want to recommend your go-to treats, SnackMatch makes it easy to connect over delicious snacks.
## Features
- User Registration and Login
- Snack Search and Discovery
- Snack Recommendations
- User Reviews and Ratings
- Snack Sharing with Friends
- Personalized Snack Lists
## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/snackmatch.git
    cd snackmatch
    ``` 2. Install dependencies:
   ```bash
   npm install
   ``` 3. Set up environment variables:
   Create a `.env` file in the root directory and add the following:
   ```
   PORT=3000
   DATABASE_URL=your_database_url
   JWT_SECRET=your_jwt_secret
   ``` 4. Start the application:
   ```bash
   npm start
   ``` 5. Open your browser and navigate to `http://localhost:3000`
